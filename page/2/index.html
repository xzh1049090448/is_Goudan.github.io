发表于2022-02-20|C++
静态类型转换（static_cast） 写法static_cast<目标类型>(原始对象)。 可以进行基础类型转换。 父与子的class可以转换。 没有父子关系的class不可以转换。 动态类型转换（dynamic_cast） 写法dynamic_cast<目标类型>(原始对象)。 不可以转换基础数据类型。 父子间可以转换。 一般父转子，不可以。 子转父，可以。 如果发生多态，都可以。 常量转换（const_cast） 写法const_cast<目标类型>(原始对象)。 用来修改类型的const属性。 常量指针和非常量指针相互转换，并且仍然指向原来对象。 常量引用和非常量引用相互转换，并且仍然指向原来对象。 ​ 注意：不能对非指针和非引用的变量使用const_cast操作符去移除他的const。 重新解释转换（reinterpret_cast） 写法reinterpret_cast<目标类型>(原始对象)。 什么都可以转，最不安全，不推荐。

友元
发表于2022-01-14|C++
友元目的​ 访问类中的私有成员属性。 三种友元 全局函数做友元函数，在类中写friend void show(); 整个类做友元类。在类中写friend class Person; 注意事项： 友元关系不能被继承。 友元关系是单向的，类A是类B的朋友，但是类B不一定是类A的朋友。 友元关系不具有传递性。类B是类A的朋友，类C是类B的朋友，但是类C和类A不一定是朋友。 成员函数做友元函数，在类中写friend void Person::show();
